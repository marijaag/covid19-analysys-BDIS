# Predviđanje i analiza kliničkih ishoda kod pacijenata zaraženih virusom COVID-19

Ovaj projekat se fokusira na primenu Big Data tehnologija u domenu **pametnog zdravstva**. Glavni cilj je identifikacija ključnih faktora rizika i predviđanje kliničkih ishoda kod pacijenata zaraženih virusom COVID-19 na osnovu njihovih demografskih karakteristika i prethodnih oboljenja.

## Uputstvo za setup projekta

Zbog veličine skupa podataka (preko 1 milion redova), izvorni podaci se **ne nalaze** na GitHub repozitorijumu.

**Nakon inicijalnog kloniranja ovog repozitorijuma potrebno je uraditi sledeće:**

1. U korenu projekta kreirajte folder pod nazivom `data/`.
2. Preuzmite dataset sa sledećeg linka:  
   [Kaggle - COVID-19 Dataset](https://www.kaggle.com/datasets/meirnizri/covid19-dataset/data)
3. Preuzeti CSV fajl (pod nazivom `Covid Data.csv`) smestite u kreirani `data/` folder.

**Napomena:** `.gitignore` je podešen da ignoriše ovaj folder kako se veliki podaci ne bi greškom otpremili na GitHub.
